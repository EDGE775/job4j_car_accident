## General info
This application is the base of auto accidents.
It is a simple system with authorization and registration of users.

## Technologies
* Java 17
* Spring (MVC, ORM, Data, Security)
* PostgreSQL
* Hibernate
* JDBC
* Apache TomCat
* HTML, Bootstrap, JSP, JSTL
* Maven
* Junit
